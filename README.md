Breve descrição do sistema: O projeto consiste em uma plataforma de estudos gamificada que utiliza inteligência artificial para gerar questões de múltipla escolha com base no tema escolhido pelo usuário. O sistema permitirá selecionar dificuldade, quantidade de fases e acompanhar pontuação, ranking e sequência de acertos, tornando o estudo mais dinâmico, rápido e eficiente para alunos do Ensino Médio.
